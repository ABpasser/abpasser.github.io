Getting Started准备开始

A-Frame 是一个创建虚拟现实网页体验的框架，它可以在桌面系统浏、移动设备以及Oculus Rift.

有几个方式可以获取A-Frame及样例：

  • 官方样例
  • npm获取
  • CDN下载 JS文件
  • CodePen中查看样例

在安装文档中查看操作细节。

要开始创建一个 A-Frame 场景，我们首先引入 A-Frame 的 JavaScript 文件，然后添加一个 <a-scene> 标签。每个 A-Frame 创建出有动画需要的方块实体都需要放置在一个 <a-sence></a-sence> 中。

<!doctype html>
<html>
  <head>
    <title>My first VR site</title>
    <script src="https://aframe.io/releases/latest/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
    </a-scene>
  </body>
</html>

我们的 A-Frame 场景已经准备好添加物件了！让我们开始添加一个简单的方块。

<!doctype html>
<html>
  <head>
    <title>My first VR site</title>
    <script src="https://aframe.io/releases/latest/aframe.min.js"></script>
  </head>
  <body>
    <a-scene>
      <a-cube></a-cube>
    </a-scene>
  </body>
</html>

当我们用浏览器打开我们的场景，我们就可以看到这个方块了！

安装

样例-初学者工具包

样例-初学者工具包是开始一个新的 A-Frame 项目的很好途径。它在一个<a-scene>标签中包含了一些基础元素，并且引入了最新版本的框架文件。有几种获取样例-初学者工具包的方式：

  1. 直接下载 zip 文件
  2. 至 CodePen 查看
  3. 至 github 查看

使用 git 的开发者可以方便快捷的克隆这个库。对只用 npm 的开发者，还包括可自动重加载的本地开发服务器（可选）（@mattdesl‘s fantastic budo）并支持browserify（可选），以及GitHub页面部署流程。

baidu

独立文件下载

如果你不需要样例，只想使用 A-Frame 框架，直接点击以下链接下载所需版本，在<script>标签中包含 A-Frame 文件即可。这样你就可以使用完整的 A-Frame 功能，并将 AFRAME 注册为全局变量。

  • 生产环境版 最小化
  • 开发版 未压缩，包含 source maps

Note: The minified version is not recommended for use during development, as it does not provide the useful warnings for common mistakes that the unminified version does.

NPM

使用 A-Frame 搭建场景时，npm 是推荐安装的方式。它对 CommonJS 规范的模块打包工具（如Webpack 或 Browserify）支持友好。

# latest stable version
$ npm install aframe

# dev build (directly from GitHub)
$ npm install mozvr/aframe#dev

AMD模块加载

独立下载的文件和通过 npm 安装的版本都包含 UMD，所以可以直接按 AMD 规范使用模块加载。
